# Termos de Uso

Última atualização: junho de 2026.

O Afinados é um produto da Eiseron. Ao usar o Afinados, você concorda com estes termos.

## O que o Afinados é

O Afinados é um conjunto de ferramentas que produz **estimativas geométricas comparativas** para apoiar o acerto de carburador. As ferramentas calculam aproximações de geometria (por exemplo, a área de passagem de combustível ao longo do acelerador) a partir de dados de catálogo.

## O que o Afinados não é

O Afinados **não mede vazão** e **não substitui** teste em bancada, dinamômetro ou pista. Os resultados são uma referência para comparar acertos, não uma recomendação de gicleagem validada. **Acerte por sua conta e risco.** A Eiseron não se responsabiliza por danos ao motor, perda de desempenho ou qualquer prejuízo decorrente do uso das estimativas.
