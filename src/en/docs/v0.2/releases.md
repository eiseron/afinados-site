# What's new in 0.2

A log of user-facing changes in the 0.2.x line.

## 0.2.1

- This release-notes page is now part of the documentation.

## 0.2.0

- **Revised fuel-passage curve:** the curve now includes the main circuit's idle contribution (it was previously measured above idle), so setups saved in 0.1 — Mikuni included — may show a slightly fuller curve.
- **Multi-manufacturer:** the carburetor manufacturer is now a first-class choice in the simulator — pick between Mikuni and Keihin.
- **Keihin support:** selectable Keihin needles and needle jet (including drop-in clones such as NIBBI, KOSO and OKO), with the N427-90 needles for FCR 28–33 mm.
- **Persistent guest session:** your setups no longer disappear when you close the browser.
