# Afinados Documentation

Afinados is a set of tools to support carburetor tuning. Each tool has its own section below.

> **Comparative estimate only, not jetting advice.** The tools give geometric estimates, not measured flow rates. Tune at your own risk.

## Tools

- [Fuel-passage area](fuel-passage-area/index.md): estimate and compare the fuel-passage area of a Mikuni or Keihin carburetor (including drop-in clones such as NIBBI, KOSO and OKO) across throttle position.

More tools are on the way.
